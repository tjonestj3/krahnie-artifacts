# Krahnie Godot Tetris

Playable Godot 4 Web export inspired by tjonestj3/rust-tetris: retro space Tetris, hold/next queue, ghost piece in Mage mode, progressive levels, lock delay, line clear flash animation, big Tetris celebration, high score persistence, and mage ghost-block spell charges.

Controls: arrows/WASD move, Up/W/X rotate, Z rotate back, Space hard drop, C hold, B cast ghost spell, L toggle Legacy mode, P pause, R restart.

Legacy mode: disables hold/spells/ghost preview and uses a slightly slower classic-feeling gravity profile.
