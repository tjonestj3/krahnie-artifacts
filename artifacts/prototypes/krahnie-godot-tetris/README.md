# Krahnie Godot Tetris

Playable Godot 4 Web export inspired by tjonestj3/rust-tetris: retro space Tetris, hold, ghost piece, progressive levels, line clear particles, high score persistence, and mage ghost-block spell charges.

Controls: arrows/WASD move, Up/W/X rotate, Z rotate back, Space hard drop, C hold, B cast ghost spell, P pause, R restart.
