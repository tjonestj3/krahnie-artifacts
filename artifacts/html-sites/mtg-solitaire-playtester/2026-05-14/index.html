A browser-side Untap-inspired sandbox for goldfishing your own Magic games. Import a decklist, let Scryfall hydrate card images/text, shuffle up, draw hands, tap cards, and manually move cards across zones. No rules engine yet — this is the fast playtest mat.

1. Import deck

Supports Untap-style text lines: 4 Lightning Bolt, 1x Sol Ring, optional set codes ignored for now. Scryfall is queried in chunks via /cards/collection.

Untap.in research → artifact plan

What Untap is:
Browser-based online TCG playtesting with opponents. It focuses on manual tabletop play rather than fully automated rules.
Deck import pattern:
Untap wiki supports simple text import formats like 1 CardName, 1x CardName, and optional (setId).
Our V1 angle:
Solitaire/goldfish first: fast deck import, Scryfall images, zones, tap/untap, counters, draw/mulligan, and a play log.
